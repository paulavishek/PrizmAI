About Webhooks

What are webhooks?

Webhooks allow external services to receive real-time notifications when events occur on your board.

Common uses:

  • Send notifications to Slack channels
  • Update MS Teams with task changes
  • Sync with external project management tools
  • Trigger custom automation workflows

Payload format:

Example — task.completed event:

{
  "event": "task.completed",
  "timestamp": "2026-04-27T10:30:00Z",
  "delivery_id": 123,
  "data": {
    "id": 42,
    "title": "Design landing page",
    "completed_by": "avishek@example.com",
    "board": {
      "id": 100,
      "name": "Software Development"
    }
  }
}
