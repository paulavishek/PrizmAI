# AI Assistant RAG & Strategic Thinking Test Questions

## üéØ Purpose

These questions test the AI Assistant's ability to:
1. **Combine web search (RAG)** with project-specific data
2. **Apply best practices** from industry to your specific situation
3. **Provide strategic recommendations** based on both external knowledge and internal data
4. **Think critically** about project management challenges

## üîç How RAG Works in These Questions

**RAG (Retrieval-Augmented Generation)** means:
1. AI detects a strategic/how-to question
2. Performs web search for current best practices
3. Retrieves your project data from database
4. **Combines both** to give contextualized advice

3. **üìä Retrieve project data** - Get relevant data from your database

## üìä Response Quality Indicators

### ‚úÖ Good RAG Response Should Include:

1. **External Knowledge**
   - "According to agile best practices..."
   - "Industry research shows..."
   - "Common strategies include..."
   - Source citations when available

2. **Your Specific Data**
   - "Based on your Software Project board..."
   - "Looking at your team of 6 members..."
   - "Your 3 overdue tasks are..."
   - Actual numbers and names

3. **Combined Recommendations**
   - "Given your situation, I recommend..."
   - "Applying [best practice] to your [specific task]..."
   - "Here's a tailored plan for your project..."

4. **Actionable Steps**
   - Numbered action items
   - Specific assignments
   - Clear priorities
   - Expected outcomes

### ‚ùå Poor RAG Response Would Be:

- Generic advice without project context
- Project data without strategic guidance
- No web search when clearly needed
- Conflicting advice from web vs data
