What is the Shadow Board?

Shadow Board lets you run multiple “what-if” scenarios as living, parallel branches of your project. Unlike the What-If Analyzer (which gives you a one-time snapshot), Shadow Board branches update automatically as real work progresses — so you’re always comparing live feasibility scores, not stale calculations. When one branch clearly wins, commit it to reality and archive the rest.

What-If Analyzer Static snapshot

A calculator. You adjust sliders, get a feasibility score, and save the result. It doesn’t know what happened on your board yesterday.

Snapshot in time. Doesn’t update. Good for quick exploration and saving interesting numbers.

Shadow Branch Living simulation

A living clone of your project running a different set of assumptions. Every time someone completes a task or a deadline moves on the real board, all branches recalculate instantly.

Live branch. Recalculates every time real work changes. Shows current feasibility, always up to date.

Recommended workflow
  1. Run a What-If scenario with the sliders
  2. Click Promote to Shadow Branch
  3. Repeat for each option you want to compare live
  4. Watch the Quantum Standup — it shows how today’s real work changed each branch
  5. Click Commit when one branch clearly wins
